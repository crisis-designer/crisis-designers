# ⚡ 위기의 디자이너들 (Crisis Designers)

AI 진화의 파도 속에서 살아남으려는 디자이너들을 위한 실시간 생존 가이드입니다. 
현재 배포 중인 주요 AI 툴들의 업데이트 내역과 그에 따른 디자이너들의 위기 지수를 공유합니다.

### 🛠 핵심 기능
- **AI Update Feed**: 최신 AI 툴의 버전 정보 및 주요 변경 사항 요약
- **Crisis Index**: 사용자 참여형 위기 지수 투표 (현재 로컬 시뮬레이션)
- **Dystopian UI**: 네온 사인과 다크 모드를 활용한 화려하면서도 공허한 디자인

### 🎨 디자인 컨셉
- **Theme**: 디스토피아 판타지
- **Colors**: Orange Neon, Black, Zinc
- **Mood**: "화려하지만 공허한 미래"
